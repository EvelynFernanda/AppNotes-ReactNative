# Notes App

## Sobre

Após o treinamento de React Native que tivemos na IBM, foi nos passado para desenvolver um aplicativo de notas, onde utilizei:
- React Navigation 
- Async Storage
- React Native CLI
- Emulador Android Studio
- IDE: Visual Studio Code
